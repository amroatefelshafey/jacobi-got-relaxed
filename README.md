0402340 - Engineering Computation & Linear Algebra Project

This is an academic, group project made by a team of four members, namely, Amro Atef, Abdelrahman Almallah, Abdelrahman Zalloum, and Rafi Azad.

The project is composed of two tasks. The first involves solving a system of linear equations derived from electrical circuit analysis via the Gauss-
Jordan Elimination Algorithm. We are also required to check for the convergence condition, which if not satisfied, will force us to reformulate the
system and then solve the new one using iterative methods.

The second task involves finding the root of a given cubic polynomial using the false position & secant methods, with initial conditions [x0=1,x1=10]

The full project is to be implemented using MATLAB and will be documented well.
